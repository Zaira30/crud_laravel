# crud_laravel
crud laravel, com validação de cpf

Alterar o arquivo .env de acordo com sua conexão

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3303
DB_DATABASE=cruds
DB_USERNAME=root
DB_PASSWORD=upnext

rodar no terminal os seguintes comandos:
php artisan key:generate 
composer update
php artisan migrate
